DATA ANALYSIS PROJECT
ANALYSIS OF RIVER TOXICITY LEVELS

SUMMARY

This data project explores the varying concentrations of toxins found in the Amazon, Danube, Mississippi, Nile and Yangtze rivers from 2018 to 2022.

Key Insights:
1. Yangtze river contains the highest concentrations of lead, mercury and arsenic of all five rivers.
2. Danube river contains the lowest concentrations of mercury and arsenic of all five rivers.
3. Strong positive correlation found between water pH level and nitrate concentration, with statistically signifant result.
4. T-test for lead concentrations observed between the Yangtze and Nile rivers returned as statistically significant.

An interactive web application was designed using Plotly Express which allows scientists to explore toxicity data for the world's largest water systems.
All plots include both a slide bar for specifying the date range and a legend which can be selected to filter data for different rivers.

CHALLENGES

missing values:
- a small number of measurements were missing from the dataset, which needed replacing in order to calculate a column average.
- I decided to use the interpolate() method to approximate the values using measurements either side.

plotly:
- using plotly I wanted to add a slider which would dynamically change the range of data displayed, as opposed to a static filter where you select from a list of years.
- I decided to include the 'rangeslider' functionality for the x-axis in plotly express, which treats its type as 'date'.

TECHNIQUES

plotly:
- plot measurements for multiple toxins in each river system (px.line(y=[df.Lead,df.Mercury]))
- save figure to html file (fig.write_html())

seaborn:
- create bar charts comparing toxin concentrations in each river (sns.barplot(df))
- change bar colours depending on x-axis variables (hue="River")

scipy:
- build a t-test module for independent testing between two samples (stats.ttest_ind())
- form null / alternative hypotheses and test significance (if p_value < 0.05: "reject null hypothesis")

EXTENSIONS

The project could be extended to include:
- an impact assessment of the effects of toxin level on aquatic life and human health.
- a root cause analysis of the variations in toxin levels and how river waste can be managed.
